웹 타이포그래피 활용

IA inc.의 창립자(CEO), Oliver Reichenstein은 웹 상의 정보는 대부분 문자를 통해 이루어지고 있기 때문에 웹 디자이너는 주어진 정보를 사용자가 잘 읽고, 이해할 수 있도록 타이포그래피 훈련을 게을리 하지 말아야 한다고 말했습니다.

물론 95%라는 출처 모를 통계에 대한 의구심은 듭니다만, 웹 디자인에 있어 타이포그래피가 차지하는 비중이 크다는 것만은 분명한 사실입니다. 그렇기 때문에 웹 디자이너는 타이포그래피를 잘 사용하기 위한 훈련이 필요합니다. 정보를 효과적으로 전달하기 위해서는 누구나 명확하게 구분할 수 있고, 읽기 좋은 ‘굿 타이포그래피(Good Typography)’ 능력이 요구되니까요.

웹 디자인을 구성하는 95%는 타이포그래피

디자이너의 타이포그래피 사용 능력은 디자인 기본기의 탄탄함을 알 수 있는 지표입니다. 그렇기에 해외 디자이너의 작품 다수는 웹 타이포그래피에 심혈을 기울여 제작합니다. 화면 상의 디자인을 마우스로 드래그하여 블록 선택해보면, 디자인이 순수 웹 타이포그래피(HTML+CSS)로 구성되어 있는 것을 확인할 수 있습니다.
